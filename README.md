[cite_start]Este projeto nasceu de uma necessidade real: tirar o controle de prazos e faltas do papel e centralizá-los em um sistema funcional[cite: 1]. [cite_start]É uma ferramenta pensada para professores que precisam gerenciar múltiplas estagiárias com eficiência[cite: 1].

[cite_start]Desenvolvi este app para aplicar os conhecimentos de Análise e Desenvolvimento de Sistemas da UNIFOR em um cenário real[cite: 1, 2].

### 🛠️ Tecnologias Utilizadas
* [cite_start]**React + Vite:** Performance e agilidade no desenvolvimento[cite: 1].
* [cite_start]**TypeScript:** Segurança de dados e código mais limpo, evitando erros de "undefined"[cite: 1, 2].
* [cite_start]**Supabase:** Backend as a Service com Auth e banco de dados Realtime[cite: 1, 2].
* [cite_start]**Tailwind CSS:** Interface moderna e responsiva focada em usabilidade[cite: 1, 2].

### 🚀 Principais Recursos
* [cite_start]**Dashboard Visual:** Status coloridos para identificar atrasos imediatamente[cite: 1].
* [cite_start]**Gestão de Prazos:** Organização centralizada de documentos e entregas[cite: 1].
* [cite_start]**Calendário Mensal:** Controle simples de faltas e horas[cite: 1].
* [cite_start]**Segurança (RLS):** Isolamento total de dados entre diferentes professores[cite: 1, 2].

### ⚙️ Como rodar o projeto
1. [cite_start]Instale as dependências: `npm install`[cite: 1].
2. [cite_start]Configure o `.env` com suas chaves do Supabase seguindo o `.env.example`[cite: 1].
3. [cite_start]Inicie o servidor: `npm run dev`[cite: 1].
